# API Testing using Postman

This project demonstrates API testing using Postman for validating REST APIs.

## 🔧 Features
- Tested GET and POST endpoints
- Validated response status codes
- Verified response payloads
- Implemented negative test scenarios
- Used environment variables for flexible configuration

## 🛠️ Tools & Technologies
- Postman
- JSONPlaceholder API

## 📌 Test Scenarios Covered
- Get list of users
- Get single user
- Get invalid user (negative test)
- Create user (POST request)
- Validate response structure and fields

## ▶️ How to Use
1. Import the collection JSON file into Postman
2. Set environment variable:
   - `base_url = https://jsonplaceholder.typicode.com`
3. Run requests individually or using Collection Runner

## 📂 Project Structure
- Postman Collection (JSON file)
